## Description
A comprehensive theme customization tool for Spotify Desktop.

## Installation
1. Copy `theme-editor.js` and `theme-editor.css` to your Spicetify Extensions folder:
   - Windows: `%appdata%\spicetify\Extensions\`
2. Enable the extension:
   ```bash
   spicetify config extensions theme-editor.js
   spicetify apply
   ```
